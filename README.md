# Site de Intercâmbio (Boston & New York)

Um site estático simples, bonito e elegante, com três páginas:
- Home (introdução e cards de navegação)
- Boston (galeria)
- New York (galeria)

## Como usar

- Basta abrir o arquivo `index.html` no seu navegador.
- Se preferir um servidor local, no PowerShell:

```powershell
python -m http.server 8080
# Abra http://localhost:8080
```

## Personalização

- Troque as imagens editando `boston.html` e `newyork.html` (atributos `src` e `alt`).
- Altere cores e tipografia em `styles.css` (variáveis CSS em `:root`).
- Edite textos da Home em `index.html`.

## Créditos das imagens

As imagens utilizam links do Unsplash para fins de demonstração.
